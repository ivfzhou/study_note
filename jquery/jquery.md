# 选择器

多个选择表达式中间逗号分割会选择他们的并集，若是接壤着会选择它们的交集。字符串中 \\ 表示 \，\ 可以转义其它特殊字符。  

- \* 选取所有元素，某些浏览器使用 * 的处理速度缓慢。
- this 选取当前对象，当前HTML元素。
- #ID名 选取为该id的元素，某些浏览器使用数字开头的ID名称可能会出问题。
- .类名 选取为该类名的所有元素，某些浏览器使用数字开头的类名可能会出问题。
- 元素 选取为该元素的所有元素。
- 元素>元素 选取所有前元素下直接子元素为后元素的所有元素。
- 元素 元素 选取所有前元素下子元素为后元素的所有元素。
- 元素+元素 选取所有前元素下紧接的兄弟元素为后元素的元素。
- 元素~元素 选取所有前元素下兄弟元素为后元素的所有元素。
- \[属性名] 选取所有有该属性元素。
- \[属性名=’属性值’] 选取有该属性且等于该值的所有元素。
- \[属性名!=’属性值’] 选取有该属性但不等于该值的所有元素。
- \[属性名^=’属性值’] 选取有该属性且属性值以该值开头的所有元素。
- \[属性名$=’属性值’] 选取有该属性且属性值以该值结尾的所有元素。
- \[属性名*=’属性值’] 选取有该属性且属性值中包含该值的所有元素。
- :first 选取DOM树从根到尾第一个元素。
- :last 选取DOM树中最后一个元素。
- :first-child 选取属于其父元素的第一个子元素的所有元素。
- :last-child 选取属于其父元素的最后一个子元素的所有元素。
- :only-child 选取属于其父元素唯一子元素的所有元素。
- :nth-child(数字) 选取属于其父元素第数字个子元素的所有元素，数字从1开始。
- :nth-last-child(数字) 选取属于其父元素倒数第数字个子元素的所有元素，数字从1开始。
- :first-of-type 选取属于其父元素第一个该类型的子元素的所有元素。
- :last-of-type 选取属于其父元素最后一个该类型的子元素的所有元素。
- :only-of-type 选取属于其父元素唯一一个该类型的子元素的所有元素。
- :nth-of-type(数字) 选取属于其父元素第数字个该类型的子元素的所有元素，数字从1开始。
- :nth-last-of-type(数字) 选取属于其父元素倒数第数字个该类型的子元素的所有元素，数字从1开始。
- :input 选取所有input元素。
- :text 选取所有type=”text”的input元素。
- :password 选取所有type=”password”的input元素。
- :radio 选取所有type=”radio”的input元素。
- :checkbox 选取所有type=”checkbox”的input元素。
- :submit 选取所有type=”submit”的input和button元素，如果button标签没有定义类型，大多数浏览器会把该元素当作类型为submit的按钮。
- :reset 选取所有type=”reset”的input和button元素。
- :button 选取所有type=”button”的input和button元素。
- :image 选取所有type=”image”的input元素。
- :file 选取所有type=”file”的input元素。
- :enabled 选取所有未禁用的input和button元素。
- :disabled 选取所有禁用的input和button元素。
- :selected 选取当前选中的下拉列表元素。
- :checked 选取所有checked=”checked”的input元素和当前选中的下拉列表元素。
- :focus 选取当前聚焦的元素。
- :has(元素) 选取包含指定元素的所有元素。
- :empty 选取所有空内容的元素，会选取单标签。
- :parent 选取所有含有子元素或者文本的元素。
- :hidden 选取所有隐藏了的元素，例如 head、mate、script。
- :visible 选取所有可见的元素。
   不可见元素：
   1.设置为display:none。
   2.type=”hidden”的表单元素。
   3.width和height设置为0。
   4.隐藏的父元素（同时隐藏所有子元素）。
- :root 选取根元素，html。
- :lang(语言代号) 选取有属性lang(lang)的元素，包含其子元素。
- :contains(文本) 选取包含指定文本的元素及其父元素。
- :target 选取触发a标签目标#id的元素。
- :even 选取从0开始偶数位的元素，不关注父子关系。
- :odd 选取从0开始奇数位的元素，不关注父子关系。
- :eq(数字) 选取元素中第数字位的元素，数字从0开始。
- :gt(数字) 选取元素中大于数字位的元素，数字从0开始。
- :lt(数字) 选取元素中小于数字位的元素，数字从0开始。
- :not(选择器) 选取非选择器选取的元素的元素，不排除子元素。
- :header 选取标题元素，标题元素为h1-h6。
- :animated 选取动画元素。

# 事件方法

- bind(（事件名,可选传入函数的JSON格式数据,函数）或者（{事件名:函数,...}）) 向元素添加事件。多个事件名空格分割。同一事件多次添加，只触发最后添加的。
- unbind(（可选事件名,可选函数名）或者（event）) 移除元素通过jQuery添加的事件。多个事件名空格隔开。不添加参数则移除该元素所有事件。
- delegate(选择器, 事件名, 可选传入函数的JSON格式数据, 函数) 向元素的子元素添加事件。
- undelegate(可选选择器, 可选事件名, 可选函数名) 移除通过delegate向子元素添加的事件。无参则移除所有。
- live(事件名, 可选传入函数JSON格式数据, 函数) 向元素添加事件。
- die(可选事件名, 可选函数名) 移除元素通过live添加的事件。
- one(事件名, 传入函数的JSON格式数据, 函数) 向元素添加事件。不过该事件只运行一次。
- click(可选函数) 触发或添加元素的单击事件。
- toggle(函数, 函数...) 向元素添加点击事件。事件轮流触发函数。
- dblclick(可选函数) 触发或添加元素的双击事件。
- mouseenter(可选函数) 触发或添加元素的鼠标进入（不含子元素）事件。
- mouseleave(可选函数) 触发或添加元素的鼠标离开（不含子元素）事件。
- hover(函数1, 函数2) 等价于mouseenter和mouseleave。
- mouseover(可选函数) 触发或添加元素的鼠标进入事件。
- mouseout(可选函数) 触发或添加元素的鼠标离开事件。
- mouesemove(可选函数) 触发或添加元素的鼠标移动事件。每移动一像素触发一次。
- mouesedown(可选函数) 触发或添加元素的鼠标按下事件。
- mouseup(可选函数) 触发或添加元素的鼠标松开事件。
- focus(可选函数) 触发或添加元素的聚焦事件。
- blur(可选函数) 触发或添加元素的失焦事件。
- keydown(可选函数) 触发或添加元素的按键按下事件。
- keyup(可选函数) 触发或添加元素的按键松开事件。
- keypress(可选函数) 触发或添加元素的按键事件。不会触发所有的键(ALT、CTRL、SHIFT、ESC)。
- load(函数) 添加元素（带有URL的元素）的加载后事件。
- unload(函数) 添加window的页面离开事件。
- change(可选函数) 触发或添加元素的改变（文本域、textarea、select）事件。
- ready(函数) 添加元素的就绪事件。
- resize(可选函数) 触发或添加浏览器窗口大小改变事件。
- scroll(可选函数) 触发或添加可滚动元素的滚动事件。
- select(可选函数) 触发或添加元素（input文本、textarea文本）的文本被选择事件。
- submit(可选函数) 触发或添加表单元素的提交事件。
- error(可选函数) 触发或添加元素的未正常加载事件。
- trigger(（事件名,可选传入函数参数,...）或者（event）) 触发元素事件。
- triggerHandler(事件名, 可选传入函数参数, ...) 触发元素事件。不会触发元素默认行为，且该方法返回事件函数的返回值。
- 事件参数中的函数的参数event对象的属性和方法：
- which 返回按键的数字值。
- key
- data 接收到的额外参数。
- currentTarget 返回当前元素。
- delegateTarget 返回当前元素。
- relatedTarget 返回当鼠标移动时哪个元素进入或退出。
- target 返回触发该事件元素。
- namespace 返回当事件被触发时指定的命名空间。
- pageX 返回相对于元素左边缘的鼠标位置。
- pageY 返回相对于元素上边缘的鼠标位置。
- result 返回最后一个事件函数返回值。
- timeStamp 返回时间触发时间距离1970-1-1的毫秒数。
- type 返回事件名。
- metaKey 事件触发时META键是否被按下。
- preventDefault() 阻止事件的默认行为，例如单击a标签打开链接，阻止对表单的提交。
- isDefaultPrevented() 返回是否调用了event.preventDefault()。
- stopImmediatePropagation() 阻止其他事件处理程序被调用。
- isImmediatePropagationStopped() 返回是否调用了。
- event.stopImmediatePropagation()。
- stopPropagation() 阻止事件向上冒泡到DOM树，阻止任何父处理程序被事件通知。
- isPropagationStopped() 返回是否调用了event.stopPropagation()。

# 动画方法

- hide(可选速度, 可选回调函数) 隐藏元素。速度单位毫秒。默认0，可选normal slow fast。
- show(可选速度, 可选回调函数) 显示元素。速度单位毫秒。默认0，可选normal slow fast。
- toggle(（可选速度,可选回调函数）或者（布尔值）) 显示或隐藏元素。速度单位毫秒默认0，可选 slow fast normal。
- fadeOut(可选速度, 可选回调函数) 隐藏元素。速度默认normal，可选solw fast，单位毫秒。
- fadeIn(可选速度, 可选回调函数) 显示元素。速度默认normal，可选solw fast，单位毫秒。
- fadeTo(可选速度, 透明度, 可选回调函数) 调准元素的透明度。速度默认normal，可选solw fast，单位毫秒。透明度值在0.00-1.00之间。
- slideUp(可选速度, 可选回调函数) 滑动隐藏元素。速度默认normal，可选solw fast，单位毫秒。
- slideDown(可选速度, 可选回调函数) 滑动显示元素。速度默认normal，可选solw fast，单位毫秒。
- slideToggle(可选速度, 可选回调函数) 滑动显示或隐藏元素。速度默认normal，可选solw fast，单位毫秒。
- animate(CSS JSON 格式字符串, 可选速度, 可选easing函数, 可选回调函数) 改变元素CSS样式。速度默认normel，可选slow fast，单位毫秒，内置easing函数有linear、swing。
- animate(CSS JSON格式字符串, 参数) 同上。参数：{speed: easing: callback: step(每一步动画执行函数): queue(是否放入动画队列): specialEasing(指定前面 css 参数中每个属性对应的 easing函数): }。
- stop(是否停止所有动画, 是否允许完成当前动画) 停止元素的动画函数。
- clearQueue(可选队列名) 清除元素还未开始的动画。队列名默认标准队列fx。
- delay(时间) 对元素动画队列中还未运行的函数设置延迟时间。
- dequeue(可选队列名) 删除元素的下一个动画函数并执行。队列名默认标准队列fx。
- queue(（可选队列名,可选新队列数组）或者（函数）) 显示元素动画队列状态或者设置元素队列。或者添加运行队列函数。
- queue().length 返回队列长度。
- jQuery.queue(元素, 可选队列名, 可选队列数组) 获取添加元素的队列。

# 元素方法

- addClass(（类名）或者（函数(选择器元素位置(从0开始),原类名)）) 向元素添加类名。多个类名中间空格隔开。
- removeClass(（类名）或者（函数(选择器元素位置(从0开始),原类名)）) 移除元素的类名。
- hasClass(类名) 检查元素是否拥有指定的类名。
- toggleClass(（类名,可选添加还是移除）或者（函数(可选选择器元素位置,可选当前类名),可选添加还是移除）) 切换元素的类名。
- after({字符串}或者{函数(选择器元素位置(从0开始))}) 向元素后添加内容。可解析标签。
- before(（字符串）或者（函数(选择器元素位置(从0开始))）) 向元素前添加内容。可解析标签。
- append(（字符串）或者（函数(选择器元素位置(从0开始),原元素内容)）) 向元素追加内容。可解析标签。
- prepend(（字符串）或者（函数(选择器元素位置(从0开始),原元素内容)）) 向元素开头插入内容。可解析标签。
- replaceWith(内容或者选择器或者函数) 替换元素内容。
- $(内容).appendTo(选择器) 向元素追加内容。可解析标签。
- $(内容).prependTo(选择器) 向元素开头插入内容。可解析标签。
- $(内容或者选择器).insertAfter(选择器) 向元素后添加内容或者将其它元素挪到其后。可解析标签。
- $(内容或者选择器).insertBefore(选择器) 向元素前添加内容或者将其它元素挪到其前。可解析标签。
- $(内容).replaceAll(选择器) 替换元素内容。
- html(可选（内容或者函数(选择器元素位置(从0开始),原元素内容)）) 返回或设置元素的内容。返回内容为第一个元素的。
- text(可选（内容或者函数(选择器元素位置(从0开始),原元素文本)）) 设置或返回元素的文本内容。会返回所有元素的组合文本内容。
- val(可选（值或者函数(选择器元素位置(从0开始),原值)）) 设置或返回元素的值。返回内容为第一个元素的。
- attr(（属性名）或者（属性名,属性值）或者（属性名,函数(选择器元素位置(从0开始),原值)）或者（JSON格式属性键值对）) 返回或设置元素的属性。
- removeAttr(属性名) 移除元素属性。
- clone(可选是否复制事件) 返回复制元素。
- detach() 移除元素。但会保留事件和附加数据。
- remove() 移除元素。该元素还保留在jQuery对象中。
- empty() 将元素内容至空。
- unwrap() 删除元素的父元素。
- wrapAll(父元素字符串或者选择器或者元素对象) 将元素继承父元素。
- wrapinner(元素字符串或者选择器或者函数) 将元素的内容用指定的元素包裹。
- get(选择器元素位置(从0开始)) 返回元素对象。
- index(可选选择器) 返回元素在集合中的位置。从0开始。
- size() 返回元素个数。
- toArray() 返回元素数组形式。
- add(（选择器,可选位置(从0开始)）或者（元素对象）) 添加元素。
- not(选择器或者函数(下标)或者元素对象) 返回删除了符合选择器或者元素对象或者函数返回 true 的元素集合。
- filter(选择器或者函数(下标)) 返回符合选择器或函数返回true的元素。
- has(选择器) 返回元素后代中有符合选择器的元素。
- slice(开始下标, 可选结束下标) 返回元素从开始到结束下标的元素。下标从0开始，负数表示从尾部偏移量。
- eq(下标) 返回指定位置的元素。负数表示从尾部偏移量。
- find(选择器或者元素对象) 查找元素后代中符合选择器或者元素对象的元素。
- andSelf() 经查找后再将元素添加进来。
- end() 返回上一次的查找前的元素。
- children(可选选择器) 返回元素符合选择器的直接子元素。
- contents() 返回元素的直接子元素，包括文本和注释。
- first() 返回第一个元素。
- last() 返回最后一个元素。
- siblings(可选选择器) 返回元素符合选择器的同胞元素。
- next(可选选择器) 返回元素符合选择器的下一个直接同胞元素。
- nextUntil(选择器1或者元素对象, 选择器2) 返回元素以下到符合选择器1或者元素对象且符合选择器2的同胞元素。
- nextAll(可选选择器) 返回从元素以下符合选择器的同胞元素。
- prev(可选选择器) 返回元素符合选择器上一个直接同胞元素。
- prevUntil(可选选择器1或者元素对象, 选择器2) 返回元素以上到符合选择器1或者元素对象且符合选择器 2 的同胞元素。
- prevAll(可选选择器) 返回从元素符以上符合选择器的同胞元素。
- offsetParent() 返回元素最近定位的祖先元素。
- parent(可选选择器) 返回元素符合选择器的直接父元素。
- parents(可选选择器) 返回元素符合选择器的父元素。
- parentsUntil(可选选择器1或者元素对象, 选择器2) 返回元素到符合选择器1或者元素对象且符合选择器2的父元素。
- closest(选择器, 可选上下文) 从元素开始往DOM树上找到符合选择器，也在上下文中的元素。
- each(函数(选择器元素位置(从0开始),元素)) 为每个元素执行函数。
- map(函数) 返回由函数返回值组成的元素。
- is(选择器或者函数) 元素符合选择器或者函数返回true，则返回true。

# CSS方法

- css(（属性名,可选（属性值或者函数(可选选择器元素位置(从0开始),可选原属性值)））或者（JSON格式键值对对象）) 返回或设置元素样式。返回第一个匹配的元素。返回属性值时属性名不是简写属性名。
- height(可选（高度值或者函数(可选选择器元素位置(从0开始),可选原高度)）) 返回第一个元素高度或设置元素高度。默认以像素为单位。
- width(可选（高度值或者可选函数(可选选择器元素位置(从0开始),可选原高度)）) 返回第一个元素高度或设置元素高度。默认以像素为单位。
- scrollLeft(可选距离) 返回或设置水平滚动条距离左边缘的距离。以像素计。
- scrollTop(可选距离) 返回或设置垂直滚动条距离顶部的距离。以像素计。
- offset(可选（{top:left}或者函数(可选选择器元素位置(从0开始),可选原位置对象)）) 返回第一个元素的位置对象或设置元素的位置。只对可见元素有效。
- offsetParent() 返回最近定位的父元素。
- position() 返回第一个匹配元素距离父元素的位置对象。以像素计。只对可见元素有效。

# AJAX方法

- serialize() 返回表单元素URL序列化的字符串。禁用、没有name属性、提交按钮、文件按钮的数据将会忽略。
- serializeArray() 返回将表单元素转化成的JSON对象数组。形如：[{name:"",value:""}...]。禁用、没有name属性、提交按钮、文件按钮的数据将会忽略。
- load(地址, 可选发送参数, 可选函数(接受数据体,状态,xhr对象)) 发送AJAX请求将返回的数据替换元素内内容。发送参数为对象时为POST请求。地址隔空格后写选择器，选择部分数据替换元素内内容。
- \$.get(地址, 可选发送参数，JSON对象或url字符串, 可选函数(接受数据体,状态,xhr对象), 可选接受数据类型) 发起GET类型AJAX请求。
- \$.post(地址, 可选发送参数，JSON对象或url字符串, 可选函数(接受数据体,状态,xhr对象),可选接受数据类型) 发送POST类型AJAX请求。
- \$.getJSON(地址, 可选发送参数，JSON对象或url字符串, 可选函数(接受数据体,状态,xhr对象)) 发起GET类型接受JSON数据的AJAX请求。发送参数中使用?表示使用jsonp格式调用异域数据。
- \$.getScript(地址, 可选函数(接受数据体,状态,xhr对象)) 发起GET类型AJAX请求，可跨域接收脚本数据并执行。
- \$.param(对象或数组, 是否浅层序列化) 返回将对象或数组URL序列化的字符串。
- \$.ajax(可选JSON格式设置参数) 发起AJAX请求。返回xhr对象。
- \$.ajaxSetup() 返回AJAX设置json对象。
- url 请求地址。
- data 发送的参数。
- type 请求类型，默认GET。
- dataType 接受数据类型。
- async 是否异步请求，默认true。
- contentType 请求内容编码类型，默认application/x-www-form-urlencoded。
- username 设置请求用户名。
- password 指定请求密码。
- processData 是否转化发送参数，默认true。
- traditional 是否浅层转化发送参数。默认flase。
- timeout 设置超时时间，默认不超时，单位毫秒。
- context 设置success中的this，默认是$.ajaxSetup()。
- ifModified 是否服务器数据改变才发送请求，默认false。
- cache 是否缓存，默认true。
- jsonp 指定jsonp回调函数名，默认callback。
- jsonpCallback 指定jsonp回调函数，默认自动生成。
- scriptCharset 脚本字符设置。
- xhr 函数返回值指定xhr对象。
- beforeSend(xhr 对象) 发送前执行函数，返回false取消发送。
- complete(xhr对象,请求类型) 请求发送后执行函数，无论请求失败还是成功都会执行。
- dataFilter(接受到数据,请求类型) 成功接受到数据执行函数过滤数据。
- success(接受数据体,状态,xhr) 请求成功回调函数。
- error(xhr,失败信息,异常对象) 请求失败时执行函数。
